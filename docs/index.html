SARD Core UI (gulp version)

This is the main Boilerplate for SARD.

Created using Gulp 4 and Bootstrap 4, it has added Bootstrap 3 styles for components that have been discontinued in BS4 but are still part of the SARD system family.

This boilerplate is intended to be a master style guide for SARD. The build process creates a single CSS file and a Single JS file, that can be linked to from external SARD systems, creating a unified set of styles and JS behaviours that span the entire SARD suit.

That's the theory, anyway.

Components that will need updating in SARD on completion.

  • All Panels to Cards
  • Toggle Switches
