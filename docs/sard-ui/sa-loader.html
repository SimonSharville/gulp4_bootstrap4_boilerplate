SA Loader

.sa-page-loader

This is a page preloader. It shoudn't matter where it sits on the page as it is designed the cover the entire screen, but appear behind the navbar, sidemenu and footer.

Code

.sa-page-loader
  .sa-page-loader-spinner
    .loader-section.section-left
    .loader-section.section-right

You can also reduce the size of the spinner by adding
.small-spinner to .sa-page-loader-spinner

Demo

To see it work

Click Here


To close it, click on the loader spinner to replicate the page having loaded.

.sa-element-loader

This is a preloader for an individual element.

The containing element will need to have the class of .with-element-loader added to it, this gives this element a position of relative, so that the loader itself can be positioned absolute to it.
The .sa-element-loader MUST sit within this containing element.

Code

DIV.with-element-loader
  .sa-element-loader
    .sa-element-loader-spinner
      .loader-section.section-left
      .loader-section.section-right

You can also reduce the size of the spinner by adding
.small-spinner to .sa-element-loader-spinner
