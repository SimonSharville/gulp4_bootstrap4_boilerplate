Below are some variations on page titles.

Default Page Title

This is constructed as follows:
.page-title
  .name Name
  .page Page
  .subtitle Subtitle

Note If you wish to wrap any part of the page title in a headline tag (h1), it should make no differnece to the styling (see variations below).

eDocuments for Mrs Bella Owen

Default Page Title in single line

This is constructed as follows:
.page-title
  %h1
    .page
              Page
      .bold Name

Page Title with Image

This is constructed as follows:
.page-title-with-image
  .page-title-photo
    Image
  .page-title
    %h3.name Name
    %h1.page Page

Page Title with State

This is constructed as follows:
.page-title-with-state
  .page-title
    .name Name
    .page Page
  .page-state
    .current-state.Draft (or .draft)
      .state Draft
