Page Titles

Build structure
.page-title-wrapper (Important)
 .page-title-photo
 .page-title (Important)
  .page-links
  .name
  .page
  .description or .subtitle
 .page-title-actions (Add if you want Actions like buttons or form elements) (Can be added below .page-title-wrapper)
     Add actions like .btn-pt-new or search and filters
 .page-state (Add if you want Page State)
  .current-state.state
   .state Draft (or whatever)
