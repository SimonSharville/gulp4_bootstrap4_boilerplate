Theme Trust Colours

Trust colours will change when you change the theme class attached to the body tag.

To test this, go to the body tag of this page and change the class to either: .theme-trust , .theme-user or .theme-admin .
The Trust colours should change to match the User or Admin colours respectively.

The Trust colours can be changed globally by amending $trust-primary-colour and $trust-secondary-colour in the sa-colours.scss file.
