Collapsable Cards/Panels/Fieldsets

Collapsable Cards, Panels or Fieldsets are single items that are collapsable. If you have multiples of they will all open or collapse individually, unlike Accordions (see bellow), that are grouped and will close when another is opened.

Add the class of .collapsable to the .card, .panel or fieldset as well as the usual collapse elements inside.

Toggle Switches

Toggle Switches are based on Checkboxes or Radios. Each has a specific purpose and each is written slighly differently.

Toggle Switches

.sa-toggle-switch is a form of checkbox.
They can be arranged either left (default) or right by using the class of .sa-toggle-switch-right instead.

Radio Toggles

.sa-radio-toggle is a set of radio buttons, when one is selected, the other is de-selected.
They can be arranged either left (default) or right by using the class of .sa-radio-toggle-right instead.

Toggle Check

.sa-toggle-check is a single checkbox, with a YES / NO Statement.
They can be arranged either left (default) or right by using the class of .sa-toggle-check-right instead.
