Checkboxes and Radios

Standard Checkboxes and Radios are by default quite small. We have created a new set, that with a little bit of extra code added, will make them stand out better.

Build Notes

Note! Bootstrap 4 for has introduced the class of .form-check This is uses as a wrapper for Checkboxes and Radios. However, some of the earlier SARD project were built in BS3, These have their own wrappers (.sa-checkbox and .sa-radio-buttons), Which if using BS4 should be added to .form-check.

Checkboxes

To create the SARD Checkboxes add .sa-checkbox to the class of .form-check, and ADD %span.checkbox INSIDE the label.
You can instead use .sa-checkbox-after to have the checkbox after the label.

Radios

To create SARD Radios add .sa-radio-buttons to the class of .form-check, and ADD .radio-button TO the label.
You can instead use .sa-radio-buttons-after to have the button after the label.

Toggle Switches

Toggle Switches are based on Checkboxes or Radios. Each has a specific purpose and each is written slighly differently.

Toggle Switches

.sa-toggle-switch is a form of checkbox.
They can be arranged either left (default) or right by using the class of .sa-toggle-switch-right instead.

Radio Toggles

.sa-radio-toggle is a set of radio buttons, when one is selected, the other is de-selected.
They can be arranged either left (default) or right by using the class of .sa-radio-toggle-right instead.

Toggle Check

.sa-toggle-check is a single checkbox, with a YES / NO Statement.
They can be arranged either left (default) or right by using the class of .sa-toggle-check-right instead.
