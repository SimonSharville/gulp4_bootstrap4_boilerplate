Checkboxes and Radios

Standard Checkboxes and Radios are by default quite small. We have created a new set, that with a little bit of extra code added, will make them stand out better.

Build Notes

Note! Bootstrap 4 for has introduced the class of .form-check This is used as a wrapper for Checkboxes and Radios. However, some of the earlier SARD project were built in BS3, These have their own wrappers (.sa-checkbox and .sa-radio-buttons), It should be possible to use either.

Checkboxes

To create the SARD Checkboxes add .sa-checkbox to the class of .form-check and add %span.checkbox to the text INSIDE the label.
You can also use .sa-checkbox-after to have the checkbox after the label.

Radios

To create SARD Radios add .sa-radio-buttons to the class of .form-check and add .radio-button to the label.
You can also use .sa-radio-buttons-after to have the button after the label.
